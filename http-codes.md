# All HTTP codes and semantic meanings
U - unofficial
## 1XX (informational response)
* 100 - continue
* 101 - switching protocols
* 102 - processing
* 103 - early hints
## 2XX (success)
* 200 - ok
* 201 - created
* 202 - accepted
* 203 - non-authoritative information
* 204 - no content
* 205 - reset content
* 206 - partial content
* 207 - multi status
* 208 - already reported
* 218 (U) - this is fine
* 226 - im used
## 3XX (redirection)
* 300 - multiple choices
* 301 - moved permanently
* 302 - found
* 303 - see other
* 304 - not modified
* 305 - use proxy
* 306 - switch proxy
* 307 - temporary redirect
* 308 - permanent redirect

## 5XX (server errors)
* 500 - internal server error
* 501 - not implimented
* 502 - bad gateway
* 503 - service unavailable
* 504 - gateway timeout
* 505 - http version not supported
* 506 - variant also negotiates
* 507 - insufficient storage
* 508 - loop detected
* 509 (U) - bandwidth limit exceeded
* 510 - not extended
* 511 - network authenticatoin required
* 529 (U) - site is overloaded
* 530 (U) - site is frozen / origin dns error
* 540 (U) - temporarily disabled
* 598 (U) - network read timeout error
* 599 (U) - network connect timeout error
